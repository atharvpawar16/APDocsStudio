# Keyboard Shortcuts

## File Operations

| Action | Shortcut |
|---|---|
| Import file | `Ctrl + O` |
| Save as PDF (all) | `Ctrl + S` |
| Save as PDF (selected) | `Ctrl + Shift + S` |
| Save as Image (all) | `Ctrl + I` |
| Save as Image (selected) | `Ctrl + Shift + I` |
| Print | `Ctrl + P` |

---

## Scanning & Batch

| Action | Shortcut |
|---|---|
| Scan (default profile) | `Ctrl + Enter` |
| Batch scan | `Ctrl + B` |
| Scan Profile 1–11 | `F2` – `F12` |
| New profile | `Ctrl + N` |
| Profiles list | `Ctrl + L` |

---

## Image Editing

| Action | Shortcut |
|---|---|
| Rotate left | `Ctrl + Shift + ←` |
| Rotate right | `Ctrl + Shift + →` |
| Flip | `Ctrl + Shift + ↓` |
| Move up | `Ctrl + ↑` |
| Move down | `Ctrl + ↓` |
| Zoom in | `Ctrl + +` |
| Zoom out | `Ctrl + -` |
