# Changelog

## v1.0.0 — 2024 (Initial Release)

### Added
- Photo editing: brightness, contrast, saturation, hue, sharpness
- Auto-deskew for tilted images
- Black & white / grayscale conversion
- Blank page detection and auto-removal
- PDF import and export (standard + searchable)
- Page merge, reorder, and compression control
- Microsoft Paint integration — Edit with Paint, auto-reload on save
- Tesseract OCR with 40+ language support
- Searchable PDF export with embedded OCR text layer
- Email export via Gmail, Outlook, Thunderbird
- Batch processing for multiple images
- Full keyboard shortcut support
- Multi-language UI (40+ languages)
- Auto-save profiles for repetitive workflows

### Fixed
- Landing page Paint section HTML tag broken (Issue #1)
- Hero section invisible on load due to animation fill-mode (Issue #2)
- Mobile layout missing hamburger menu and grid overflow (Issue #3)
