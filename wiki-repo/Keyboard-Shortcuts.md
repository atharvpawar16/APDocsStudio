# Keyboard Shortcuts

## File Operations

| Action | Shortcut |
|---|---|
| Import file | `Ctrl + O` |
| Save as PDF (all) | `Ctrl + S` |
| Save as PDF (selected) | `Ctrl + Shift + S` |
| Save as Image (all) | `Ctrl + I` |
| Save as Image (selected) | `Ctrl + Shift + I` |
| Print | `Ctrl + P` |

---

## Batch Processing

| Action | Shortcut |
|---|---|
| Batch process | `Ctrl + B` |
| New profile | `Ctrl + N` |
| Profiles list | `Ctrl + L` |

---

## Image Editing

| Action | Shortcut |
|---|---|
| Rotate left | `Ctrl + Shift + ←` |
| Rotate right | `Ctrl + Shift + →` |
| Flip | `Ctrl + Shift + ↓` |
| Move up | `Ctrl + ↑` |
| Move down | `Ctrl + ↓` |
| Zoom in | `Ctrl + +` |
| Zoom out | `Ctrl + -` |
| Undo | `Ctrl + Z` |

---

## OCR & Tools

| Action | Shortcut |
|---|---|
| Run OCR | `Ctrl + Alt + O` |
| Email PDF (all) | `Ctrl + E` |
| Email PDF (selected) | `Ctrl + Shift + E` |
| About | `F1` |
