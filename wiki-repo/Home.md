## 🚀 Quick Start

1. Download `APDocsStudio.exe` from [Releases](https://github.com/atharvpawar16/APDocsStudio/releases)
2. Install [.NET 9 Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/9.0) if not already installed
3. Run `APDocsStudio.exe` — no installation needed
