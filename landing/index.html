Paint Integration Guide

How to edit photos using Paint's Select & Cut

Microsoft Paint is built into every Windows PC. APDocs Studio lets you open any image in Paint, edit precisely, and bring the result back as a polished PDF.

✂️ Rectangular Select & Cut

  1. Right-click a page → Edit with Paint
  2. Click Select in Paint's Home toolbar
  3. Choose Rectangular selection
  4. Draw a box around the area to remove
  5. Press Delete or Ctrl+X
  6. Press Ctrl+S — reloads instantly

🔷 Free-form Select

  1. Click Select → Free-form selection
  2. Draw around the exact shape to keep
  3. Press Ctrl+C to copy
  4. Press Ctrl+N for new canvas
  5. Press Ctrl+V to paste
  6. Save → export as PDF

🧹 Remove a Watermark

  1. Open image in Paint via APDocs Studio
  2. Set background to white (right-click Color 2)
  3. Use Rectangular Select to box the watermark
  4. Press Delete — fills with white
  5. Save with Ctrl+S
  6. Export as a clean PDF

📐 Crop to Region

  1. Draw a Rectangular selection around the region to keep
  2. Click Image → Crop in the toolbar
  3. Or press Ctrl+Shift+X (Windows 11)
  4. Save with Ctrl+S
  5. APDocs Studio picks it up automatically
  6. Export as PDF or image in one click
How it works

From photo to perfect PDF in 4 steps

No complicated setup. Open, edit, and export in minutes.

01

Open your file

Import any photo, image, or PDF — JPG, PNG, BMP, TIFF, or multi-page PDF. Drag and drop or use Ctrl+O.

02

Edit

Adjust brightness, contrast, crop, rotate, deskew. Or send to Paint for precision Select & Cut editing.

03

Run OCR

Hit Ctrl+Alt+O to extract text from any page. Creates a searchable PDF with an invisible text layer.

04

Export & Share

Save as PDF or image, email directly, or print. Auto-save profiles handle repetitive workflows.
